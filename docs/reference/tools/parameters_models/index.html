model_dump​

def model_dump(use_np_array: bool = True, *args, **kwargs)

Override the model_dump method to convert lists to NumPy arrays (if use_np_array is True) and Enums to their values.

Arguments

  • use_np_array (bool, optional): Convert lists to NumPy arrays (default is True)
  • *args: Positional arguments to pass to the parent class
  • **kwargs: Keyword arguments to pass to the parent class

Returns

  • dict: A dictionary of the model's data
